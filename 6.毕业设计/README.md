# 毕业设计项目

## 项目介绍
```txt
本项目主题为跨模态目标检测，使用语言为python，详细内容请见实验文档和实验代码文件。
```

## 项目背景
```txt
本课题面向全时段复杂场景，探索基于YOLOv8的可见光-红外多模态融合目标检测方法，具有重要的理论价值。首先，多模态融合涉及特征表示、跨模态对齐、互补信息挖掘等基础科学问题，深入研究这些问题有助于丰富多模态信息处理的理论体系。其次，现有融合方法多侧重于提升图像视觉质量，而面向检测任务的融合策略研究尚不充分，本课题探索任务驱动的融合优化方法，可为后续研究提供新思路。最后，YOLOv8作为当前主流的一阶段检测器，将其扩展至双模态输入并优化融合结构，对推动检测器架构创新具有积极意义。
```

## 主要研究问题
```txt
1.复杂场景下多模态特征的鲁棒表示问题。
2.光照感知的自适应融合机制设计问题。
3.非对齐图像的跨模态特征对齐问题。
```

## 采用的研究方法
```txt
本课题以YOLO系列为基础框架，构建双模态输入的目标检测网络，拟采用的具体方法如下：
1.引入CBAM注意力模块。
2.优化损失函数WIoU。
3.设计跨模态注意力机制。
```
